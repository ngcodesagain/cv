Day 3 (2 hours)

  • Made a commit directly from VSCode
  • Adding a CSS style tag to set style in website
  • Learned about internal CSS and external CSS
    • External CSS allows website-styling settings occur in one place
    • External CSS reduces repetition of code
  • Debugged 2 files to display the required fonts and styles, as well as, fixed the image source
  • Syntax basically means grammar
    • selector{property: value;}
    • selector: "Who is the target?"
      • Use .{className} to style selected class instead of styling the whole tag
      • Use #{idName} to style selected ID name, best for one specific part
      • Classes vs ID: There can be multiple classes but there can only be one ID
      • Pseudo-classes have semicolon ";" symbol in front of their names
        • It is possible to style differently according to the state of the website
        • Example: Different states meaning is the cursor hovering over or is the text being highlighted etc
    • property: "What characteristics do you want to change?"
    • value: "How do you want to change it?"
    • Always end with a semicolon ";" symbol
