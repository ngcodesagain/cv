Day 2 (2 hours)

  • Learned the meaning of Deprecated
    • HTML is for structure
    • CSS is for styling
    • JavaScript is for functions
  • Learned about table and table structures
    • Table has thead, tbody, tfoot
    • tr (table row) is used to add rows
    • th (table head) is used to add headers and could be added under the thead tag
    • td (table data / table cell) is used to add columns in each row
  • Learned how to use HTML tables as layout
    • cellpadding is used to set the space between each cells and its border
    • cellspacing is used to set the spacing between the cells
    • Using HTML to make a layout is NOT a great idea but it is a good practice to learn about table structures
  • Learned about HTML forms
    • label is a caption for an item
    • input type is a way to create interactive control for web-based forms in order to accept data from the user
